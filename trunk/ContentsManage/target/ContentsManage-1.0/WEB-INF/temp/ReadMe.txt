このソフトウェアの概要
======================

Lhaplus ( "ラプラス" と読みます ) は、
アーカイバと呼ばれる解凍、圧縮ソフトの一つです。

比較的、高機能な部類に入りますが、インストール直後からそのまま使用でき、
初心者用の項目が少ない設定画面も用意されているため、
PC に習熟していない方にもおすすめできるソフトウェアです。

・解凍、圧縮に、外部 DLL を導入する必要がありません
・多数のアーカイブ形式に対応しています
・様々な圧縮、解凍操作に対応し、直感的、効率的に作業を行うことができます
・ユーザーの好みに合わせて、動作や表示の設定を変更できます
・アーカイブ変換や ZIP パスワード探索など、
　アーカイブ操作に付随する作業をサポートする機能を有しています

ソフトウェアの使用法に関しては、同梱のヘルプファイルを参照してください。


再配布
======

このソフトウェアを頒布または転載するときは以下の点を遵守してください。

・オリジナルのアーカイブファイルのまま、頒布または転載してください。
　アーカイブファイルおよび格納ファイルを改変してはなりません。 

・頒布または転載の際に、メディア代、通信代等の実費を除いて、
　金銭の授受があってはなりません。 

・商業的目的で頒布する際には、事前に著作者の許可を取る必要があります。
　なお、下記の「雑誌への掲載収録」に当てはまる場合に限り、許可は不要です。

・アルファ版やベータ版など、このソフトウェアがテスト公開バージョンの場合は、
　再配布できません。 


雑誌への掲載収録
================

このソフトウェアは雑誌へ掲載、収録を歓迎しております。
下記の条件が守られている場合に限り、
著作者への連絡や見本誌の送付は必要はありません。

なお、雑誌ではなく書籍形態のものについては、
事前に著作者の許可を取る必要があります。

・オリジナルのアーカイブファイルのまま収録すること 
・最新のバージョンを掲載、収録すること (可能な範囲内でかまいません)

========

Lhaplus の配布や導入による対価で利益を得る行為は禁じておりますが、
業務を行う上での一ツールとして使用することに関しては、
各個人で使用する際の規定に準じ、特別、使用に制約や費用は発生しません。

また、ソフトウェアを社内で利用するために、
イントラネット上に Lhaplus のセットアップファイルを用意したり、
メールなどで再配布することも可能です。(「再配布」の規定に準じます )

ただし、個人や業務など、使用する用途、状況にかかわらず、
